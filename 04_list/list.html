Visual Studio Code 설치법

  1. 브라우저에 VS Code를 검색한다.
  2. 검색 결과 가장 위에 사이트에 접속한다.
  3. 자동으로 표시되는 운영체제에 맞는 다운로드 버튼을 누른다.
  4. 설정을 변경하지 않고 계속 다음을 눌러 설치를 한다.ㄴ
HTML
HTML은 HyperText Markup Language의 줄임말입니다.
CSS
CSS은 Cascading Style Sheet의 줄임말입니다.
Javascript
Java는 브라우저에 작동하는 스크립트 언어입니다.
